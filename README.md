# 🏆 my-rewards-system
**〜 自分の力で成長し、報酬を勝ち取るための公式ルールブック 〜**

このシステムは、君が「自分で自分をコントロールする力（自律心）」を磨き、日々の努力を成果に変えるためのプロジェクトです。
あなたの両親は、あなたの頑張りを全力で応援し、このルールに基づいて正当な報酬を支払います。

---

## ⚔️ 1. 朝のスタート・クエスト
**「1日の始まりを自分の力でつかみ取ろう」**

* **ミッション：** 登校前に早起きをし、自分で決めた学習課題を終わらせる。
* **報酬：** **50円 / 1日**
* **目的：** 受験生の基本である「朝のセルフマネジメント」を習慣にするため。

---

## 📝 2. 学校テスト・クエスト
**「基礎を完璧にし、ミスから学ぼう」**
*対象：カラーテスト、漢字テスト*

* **基本報酬：** **点数と同じ金額**を支給（例：90点＝90円）。
* **【完全攻略ボーナス】：** **100点満点なら、さらに ＋100円（合計200円）**。
    * ケアレスミスをゼロに抑える集中力を評価します。
* **【復活（リカバリー）報酬】：** **＋20円**
    * 100点以外でも、「なぜ間違えたか（原因）」を自分で見つけ、「次はどうするか」を説明できれば支給されます。

---

## 📊 3. 通知表・ランクアップ報酬
**「日々の積み重ねによる総合力を評価」**

* **ミッション：** 通知表で最高評価（Aなど）を獲得する。
* **報酬：** **1つにつき 1,000円**。
* **目的：** 提出物や授業態度など、半年間の誠実な取り組みを称えるため。

---

## 🎓 4. 検定試験・チャレンジ
**「高い目標への挑戦を全力サポート」**

* **ミッション：** 指定の検定試験に合格する。
* **報酬：** **受検料と同じ金額をキャッシュバック！**
* **受給額の目安（2025-2026年度データ）：**
    * 漢字検定 5級（小学校卒業レベル）：**3,000円**
    * 漢字検定 4級・3級（中学校レベル）：**3,500円**
    * 数学検定 5級（中学校1年生レベル）：**4,300円**
    * 数学検定 4級（中学校2年生レベル）：**4,300円**
    * 数学検定 3級（中学校3年生レベル）：**4,900円**

---

## 🛡 運用上の大切なお約束
1.  **自己申告制：** テストや朝の達成は、自分からパパ・ママに「報告と説明（プレゼン）」をしてください。
2.  **リカバリーの精神：** 失敗（寝坊やミス）をしても、それを分析して次に活かせば、それは「前向きな成長」として認められます。
3.  **使い道の自由：** 稼いだ報酬の使い道は自由です。自分の成長につながる「自己投資（本や文房具など）」はいつでも相談に乗ります。
